## To execute client-server program

1. CD into the correct folder using the terminal in which buyer.py and seller.py are located
2. Execute the script 'seller.py' first in the terminal using: python seller.py
3. You will then be prompted to give a name for your seller and choose a port
4. Proceed to execute the script 'buyer.py' in the terminal using: python seller.py 
5. you will then be prompted to give a name for your buyer and choose a port in which your seller is in

**YOU CAN CREATE AS MANY SELLERS JUST AS LONG AS IT'S A DIFFERENT PORT FOR EACH SELLER, YOU CAN ALSO CREATE MULTIPLE BUYERS AND HAVE THEM JOIN ONE SELLER AT A TIME. ONE SELLER CAN HAVE MULTIPLE BUYERS**
